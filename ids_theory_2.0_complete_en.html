This perspective explains the continuous transition from matter to life to consciousness through a single principle: IDS activation.

A.1 Proof of Context-Dependent Sign Reversal

Theorem: For any non-zero deviation $\Delta$, there exists a context that reverses the sign of its interpretation.
Consider the interpretation function $\mathcal{E}(\Delta, c)$. For the same physical deviation $\Delta$ (e.g., increased heart rate), define two different contexts:
  • $c_1$: Context suggesting progress toward goal achievement (e.g., during exercise).
  • $c_2$: Context suggesting deviation from safety (e.g., at rest).
In these contexts, interpretations can be modeled as:
$$\mathcal{E}(\Delta, c_1) = +|\Delta| \cdot \hat{n}_1 \text{ (positive interpretation)}$$ $$\mathcal{E}(\Delta, c_2) = -|\Delta| \cdot \hat{n}_2 \text{ (negative interpretation)}$$
Therefore, $\text{sign}(\mathcal{E}(\Delta, c_1)) = -\text{sign}(\mathcal{E}(\Delta, c_2))$, proving that the sign of interpretation for the same deviation reverses depending on context.

A.2 Proof of Scale-Free Properties

Theorem: IDS exhibits scale-free (self-similar) behavior across hierarchies.
Let the IDS functional at scale $s$ be $\mathcal{F}_{\text{IDS}}(s)$:
$$\mathcal{F}_{\text{IDS}}(s) = \int S[\Delta(s), C(s), H(s)] ds$$
Applying scale transformation $s \rightarrow \lambda s$:
$$\mathcal{F}_{\text{IDS}}(\lambda s) = \int S[\Delta(\lambda s), C(\lambda s), H(\lambda s)] d(\lambda s)$$
Assuming self-similarity of error patterns, context, and inter-hierarchical coupling:
$$\mathcal{F}_{\text{IDS}}(\lambda s) = \lambda^\alpha \int S[\Delta(s), C(s), H(s)] ds = \lambda^\alpha \mathcal{F}_{\text{IDS}}(s)$$
This power-law relationship confirms the system's scale-free properties.

A.3 Proof of Consciousness Emergence Conditions

Define total integration degree $\Phi_{\text{total}}$ using mutual information $\mathcal{I}_{ij}$ and hierarchical distance $d_{ij}$ between different components $i,j$ in the system:

C.1 Context Reversal Experiment

class ContextReversalExperiment:
    """
    Test context-dependent interpretation of identical stimuli
    """
    def __init__(self, ids_model):
        self.model = ids_model
        self.contexts = self._define_contexts()

    def _define_contexts(self):
        """
        Define contexts that assign contrasting meanings to identical stimuli
        """
        return {
            'reward_context': {'expectation': 'positive_outcome', 'valence': 0.8},
            'punishment_context': {'expectation': 'negative_outcome', 'valence': -0.8}
        }

    def run(self, stimulus: torch.Tensor):
        """
        Apply identical stimulus in different contexts and measure interpretation sign reversal
        """
        results = {}
        for context_name, context in self.contexts.items():
            self.model.hial_system.set_context(context)
            output = self.model({'data': stimulus})
            results[context_name] = {'error_interpretation': output['error']}
        
        # Calculate percentage of elements with reversed interpretation signs
        sign_reversal_tensor = torch.sign(results['reward_context']['error_interpretation']) != \
                               torch.sign(results['punishment_context']['error_interpretation'])
        
        sign_reversal_percentage = sign_reversal_tensor.float().mean().item() * 100
        return {'sign_reversal_percentage': sign_reversal_percentage}

H. Alternative Cosmological Models Considered

To assess the validity of the IDS-GUT framework, we conducted comparative verification with multiple alternative models. These include models using specific irrational numbers other than the golden ratio (φ) (e.g., $\sqrt{2}$, $\sqrt{3}$) as control ratios, as well as models that treat the ratio as a free parameter estimated from data using maximum likelihood.

Conclusion: In model comparison using 8-bin RSD data, the φ model showed superior fit (lower χ²) compared to alternative fixed-ratio models. Additionally, the maximum likelihood value obtained from the free parameter model, $r_{MLE} = 1.65 \pm 0.08$, is statistically consistent with φ=1.618..., suggesting that the data naturally selects φ. These results justify the special treatment of φ in IDS-GUT.

I. Glossary of Key Terms

IDS (Integrated Debug System):
A self-referential loop inherent in all systems that detects, interprets, and corrects deviations between the system's state and its intrinsic regularities (predictions).
Edge of Chaos:
The dynamic critical region where systems exist between completely static order and unpredictable chaos, enabling complex adaptive behavior. Life and consciousness are thought to possess active IDS in this region.
Prediction:
A broad concept in IDS theory. Refers not only to higher cognitive activities but to intrinsic state transition rules of systems at all scales, from quantum wave function evolution to biological homeostasis maintenance.
Qualia Vector ($\vec{Q}$):
A vector mathematically representing subjective experience. Consists of skeleton components forming the universal framework of experience and texture components providing sensory modality-specific qualities.
TDNG (Time-Dependent Norm Generation):
A mechanism where norms (rules) that systems should follow are dynamically generated from past, present, and future states.
HIAL (Hierarchical Integrated Adaptive Layer):
A dynamically forming and dissolving hierarchical structure of systems that processes information at different scales.
LocalDOC (Localized Dynamic Optimization Core):
An information processing hub that adaptively modifies system response rules based on error interpretation.
CIE (Critical Information Event):
A significant change in information structure requiring an adaptive response from the system.
